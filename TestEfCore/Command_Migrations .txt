# Exemple de commandes Entity Framework en markdown

Pour créer une nouvelle migration, utilisez la commande suivante :

\```
Add-Migration NomDeLaMigration
\```

Pour appliquer les migrations à la base de données, utilisez la commande :

\```
Update-Database
\```

Pour spécifier une migration particulière à appliquer, utilisez :

\```
Update-Database -Migration NomDeLaMigration
\```

Cela vous permettra de formater proprement les commandes Entity Framework dans un document markdown.
